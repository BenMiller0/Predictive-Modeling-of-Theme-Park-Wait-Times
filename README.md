# Predictive Modeling of Theme Park Wait Times

## 🛠️ Tech Stack
* **Languages/Libraries:** Python, Pandas, NumPy, Scikit-Learn
* **Automation:** Selenium, BeautifulSoup4 (Scraped 730+ coasters from RCDB & Thrill Data)

## 🚀 Key Engineering Highlights
* **Machine Learning:** Implemented machine learning models to quantify the impact of physical telemetry on guest demand.
* **Web Scraping:** Developed a robust pipeline to aggregate data across 80+ theme park domains, handling mixed units (Metric/Imperial) and request throttling.
* **Data Integrity:** Built a validation suite to identify and correct "0-minute" outliers and missing structural attributes.
* **Analysis:** Disproved the hypothesis that physical stats predict demand; found that Intellectual Property (IP) and theming are the primary drivers of wait times.
